# gowatch

a command tool written in go to watch scoeps of HackerOne
- features:
    - get new programs
    - get new added scopes
    - notify in discord

## configuration
you should add a `.env` file, including your discord `Bot_Token` and your channelID `ChannelId_general`.

```
Bot_Token="OTM***********YBC5k"
ChannelId_general="9********5"
```
